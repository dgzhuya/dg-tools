## 错误码说明
- 错误码第一位为错误类型
  | 错误码 | 类型     | 说明           | 使用场景 |
  | ------ | -------- | -------------- | -------- |
  | 1      | fs       | 文件读取错误   |          |
  | 2      | path     | 路径查找错误   |
  | 3      | exec     | 命令执行错误   |
  | 4      | network  | 网络错误       |
  | 5      | external | 外部依赖等错误 |

- 错误码第二位数字为消息需要拼接的参数个数
