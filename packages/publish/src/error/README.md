## 错误码说明
- 错误码第一位为错误类型
  | 错误码 | 类型     | 说明                   | 使用场景 |
  | ------ | -------- | ---------------------- | -------- |
  | 1      | io       | 文件或路径读取查找错误 |
  | 2      | external | 外部依赖等错误         |
  | 3      | exec     | 命令执行错误           |
  | 4      | network  | 网络错误               |
  | 5      | other    | 未定义错误             |

- 错误码第二位数字为消息需要拼接的参数个数
